**Proyecto local con Vite** 

Esta versión está preparada para correr localmente sin depender de Base44 para autenticación ni para el arranque de Vite.

**Prerequisitos:** 

1. Clone the repository using the project's Git URL 
2. Navigate to the project directory
3. Install dependencies: `npm install`

Run the app: `npm run dev`

No `.env.local` is required for the current local setup.
